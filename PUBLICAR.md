# Publicación de FranQuestions

Esta carpeta contiene únicamente los componentes necesarios para publicar el Observatorio.

- Archivo principal: `streamlit_app.py`
- Base de datos pública: `franquestions.db`
- Dependencias: `requirements.txt`
- Configuración visual: `.streamlit/config.toml`

No contiene credenciales del BCCR, preferencias personales, archivos fuente descargados, respaldos ni registros locales.

En Streamlit Community Cloud, seleccione este repositorio, la rama principal y `streamlit_app.py` como archivo de entrada.
